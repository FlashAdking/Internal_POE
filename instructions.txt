How to setup git in Windows to clone the project locally with minimal steps (no login required for public repos):

1. Download Git for Windows from: https://git-scm.com/download/win
2. Install it using the default settings.
3. Open "Git Bash" or "Command Prompt" on your Windows machine.
4. Clone the project using the HTTPS URL (No login needed!):
   git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
   
5. Enter the project folder:
   cd YOUR_REPOSITORY

6. Setup your single virtual environment:
   python -m venv venv
   venv\Scripts\activate

7. Run the projects (refer to the `script.txt` inside each folder for the exact run commands).
